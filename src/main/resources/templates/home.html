Purpose

The purpose of this demo is to provide an example implementing Spring Security with the following stack:

  • Spring Boot
  • Spring Security
  • Thymeleaf
  • JPA
  • H2 embedded database

Things to note in this demo:

  1. The top nav displays either Login or Logout and the username depending whether or not there is currently a user who authenticated successfully.
  2. The addition of the Admin item in the navigation, if the user has the admin role.
